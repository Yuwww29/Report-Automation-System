📋 Usage Instructions

Upload data file and the system will automatically:

  • Check if your uploaded data data already exists in the database: Selecting “Overwrite Existing” will replace the existing data in the database with data from the uploaded file after it has been processed
  • Check for any new names in the uploaded file that is not stored in the database: Assign them to either Commandos or ST before processing can begin
  • Process & generate PLB Tabulation Report for Commandos & STs
  • Estimated processing time: daily data – up to 30 seconds; weekly data – up to 3 minutes
  • The PLB Tabulation Report will become downloadable after processing is complete
  • The generated report will be stored on the “Processed Files” page

⚠️ Important Notes

  • Maximum upload file size: 1.5MB
  • System processes only the first sheet of the Excel File. Please verify that data is complete and accurate
  • If the file fails to process (e.g., an error occurs), refresh the page and try again. Select "Overwrite Existing" when retrying to prevent conflicts with the database
  • Do not refresh the page when the report is being generated
  • If the system crashes, refresh the page and try again after a few minutes
